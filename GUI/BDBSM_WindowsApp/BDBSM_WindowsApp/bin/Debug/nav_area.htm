Использование области навигации

Ищите способ отслеживания объектов базы данных и управления ими? Области навигации - это основной способ просмотра и доступа ко всем таблицам базы данных и она по умолчанию отображается в левой части окна.

1. Использование области навигации

        При нажатии на объект из списка, он откроется в рабочей области.

2. Быстрый поиск

        Чтобы найти нужный объект по имени, введите его имя в поле над областью.

3. Взаимодействие с объектами

        Чтобы взаимодействоваать с объектами нажмите на ПКМ по объекту.

4. Взаимодействие с областью навигации

        Чтобы взаимодействовать с областью навигаций нажмите по ней ПКМ.
