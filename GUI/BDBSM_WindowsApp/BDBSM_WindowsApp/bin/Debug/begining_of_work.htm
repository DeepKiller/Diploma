Начало работы

Чтобы создавать базы данных с помощью BDB, не нужно быть специалистом и писать код.

Создание базы данных

1. Откройте BDB.exe

    Если приложение уже открыто, нажмите кнопку "Создать новую БД"

2. Введите им базы данных, выберите расположение, а затем нажмите кнопку "Создать".

3. Введите пароль для базы данных, чтобы защитить свои данные от нежелательного доступа, а затем нажмите кнопку "ОК".
