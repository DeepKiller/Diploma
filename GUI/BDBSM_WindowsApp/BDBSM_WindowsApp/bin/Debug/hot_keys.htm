Сочитание клавиш

Сочитание клавиш помогают быстрее выполнять определенные команды.

  Примечание:

  • В сочетаниях, где одновременно нажимается несколько клавиш, их названия разделяются знаком плюс (+).

1. Сохранение базы данных

        Ctrl + Shift + S

2. Сохранение таблицы

        Ctrl + S

3. Копирование таблицы

        Ctrl + C

4. Вставка таблицы

        Ctrl + V

5. Открыть справку

        F1

6. Закрыть приложение

        Alt + F4
